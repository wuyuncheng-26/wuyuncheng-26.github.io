### 小知识

> Python里输出的内容可以更改颜色

可以运用print("\033[显示方式;字体颜色;背景颜色 + m + 内容 + \033[0m")来做到这样的效果

比如说print("\033[1;33;44mtest\033[0m")就可以显示出高亮且字体颜色为黄色，背景颜色为蓝色的test
